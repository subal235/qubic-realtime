# TurboAuth

> High-performance authentication and authorization service for Qubic blockchain

Part of the [Realtime](../../README.md) monorepo.

## Overview

TurboAuth provides wallet-based authentication and authorization for Qubic applications.

**Features**:
- Dual Protocol Support (HTTP REST + gRPC)
- Hexagonal Architecture
- L1/L2/L3 Caching Strategy
- Smart Contracts on Qubic
- Production-ready monitoring

## Quick Start

```bash
cd backend
go run ./cmd/api
```

See the [main README](../../README.md) for full documentation.
